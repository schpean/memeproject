1. Cerințe Preliminare

  • Docker instalat pe mașina locală
  • Cont Docker Hub (sau alt registru Docker)
  • Instanță AWS EC2 cu Docker instalat

2. Dezvoltare Locală

2.1 Construirea Imaginii

# Construiește imaginea local
docker build -t yourusername/memewebsite:latest .

2.2 Testare Locală

# Pornește containerele
docker-compose up -d

# Vizualizează log-urile
docker-compose logs -f

2.3 Mediul Local

  • Interfață web: http://localhost:1337
  • Endpoint-uri API: http://localhost:1337/api/*
  • WebSocket: ws://localhost:1337/ws

3. Configurare Administrator

3.1 Conectare la Baza de Date

# Conectare la containerul PostgreSQL
docker exec -it memewebsite-postgres-1 psql -U ubuntu -d meme_db

3.2 Setare Rol Administrator

# Actualizare rol utilizator la administrator
UPDATE users SET role_id = 3 WHERE email = 'mateas.bogdan@gmail.com';

# Verifică modificarea
SELECT u.email, u.username, r.name as role 
FROM users u 
JOIN user_roles r ON u.role_id = r.id 
WHERE u.email = 'mateas.bogdan@gmail.com';
Notă: După executarea acestei comenzi, trebuie să te deconectezi și să te reconectezi în aplicație pentru a activa drepturile de administrator.
Important: Asigură-te că actualizezi variabilele de mediu în docker-compose.yml pentru producție: 
REACT_APP_API_BASE_URL=https://bossme.me
REACT_APP_CLIENT_BASE_URL=https://bossme.me
CLIENT_BASE_URL=https://bossme.me
